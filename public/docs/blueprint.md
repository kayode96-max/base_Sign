# **App Name**: DexMail

## Core Features:

- Dual Authentication: Users can log in using either traditional email/password or wallet signature, enhancing accessibility.
- Crypto Transfer via Email: Allows users to send cryptocurrencies to any email address by simply including a specific syntax in the email body. The app tool then parses and processes these instructions.
- Automatic Smart Contract Wallet: Generates a smart contract wallet deterministically for recipients without existing wallets, enabling seamless onboarding.
- Email Client Integration: Supports SMTP/IMAP protocols, ensuring compatibility with various email clients like Gmail, Outlook, and Thunderbird.
- Decentralized Message Storage: Stores email messages on IPFS, indexed on the Base blockchain, providing censorship resistance and data integrity.
- Web Client Interface: Provides a user-friendly web interface for composing emails, viewing inboxes, and managing crypto transfers.

## Style Guidelines:

- Primary color: Electric blue (#0267FF) to convey a modern, trustworthy feel, aligning with blockchain aesthetics.
- Background color: Light grey (#F3F5F7), almost white, provides a clean and modern backdrop, ensuring readability and a sense of spaciousness.
- Accent color: Dark charcoal (#2C2A2A), almost black, adds depth and sophistication, primarily used for text, icons and important UI elements to ensure maximum readability and focus.
- Font: 'Kumbh Sans', a sans-serif font offering clean lines and excellent readability, suitable for both headlines and body text. Note: currently only Google Fonts are supported.
- Use minimalist icons with rounded edges to complement the modern aesthetic and ensure clarity. Prefer single-color icons in the dark charcoal.
- Maintain a clean and organized layout with ample whitespace, following a grid system for consistent alignment.
- Use subtle animations and transitions for feedback and visual interest, but avoid excessive animations.
